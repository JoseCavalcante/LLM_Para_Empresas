# Departamento de Marketing

<p>Esta projeto está inegrado à uma interface gráfica para que o usuário consiga preencher os campos até todo o processo de execução de execição da LLM e a visualização do resultado. A idéia é criar uma interface gráfica para a equipe de marketing da empresa, com objetivo de criar conteúdo personalizado</p>

## Tecnologias Aplicadas - STACKS
<p>Provedores de LLMs que oferecem acesso a modelos de IA por intermédio de APIs ou plataformas hospedadas:</p>

<ul>
  <li>Groq</li>
  <li>OpenAI</li>
  <li>Hugging Face</li>
</ul>

<p>Frameworks de orquestração que coordenam diferentes componentes facilitando a integração entre modelos e fluxos de trabalhos:</p>

<ul>
  <li>LangChain</li>
  <li>LlamaIndex</li>  
</ul>

<p>Bibliotecas para FrontEnd:</p>

<ul>
  <li>Streamli</li>
</ul>

<p>Banco de Dados Vetorial que armazena vetores para busca semantica em dados textuais</p>

<ul>
  <li>Faiss</li>
  <li>Croma DB</li>  
</ul>

<p>Ferramentas e conceitos:</p>

<ul>
  <li>Python</li>
  <li>Groq</li>
  <li>MS VS Code</li>
  <li>LangChain</li>
  <li>Técnicas de Engenharia de Prompts</li>
  <li>Streamlit</li>
</ul>
