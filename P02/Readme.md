# Atendimento e Suporte

<p>Neste projeto vamos simular o atendimento de um banco digital, no qual o usuário poderá fazer perguntas espec´ficas relacionadas a sua conta corrente</p>

<p>Trabalharemos neste projeto com um conceito chamado de RAG que vai promover documentos adicionais para ajudar na resposta da LLM. As respostas deste algoritmo serão baseadas em um documento (Manual de Atendimento) especifico disponibilizado pelo banco.</p>
